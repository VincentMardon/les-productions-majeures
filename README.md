# 🎭 Les Productions Majeures

> 💡 Laboratoire artistique indépendant — plateforme, musique et œuvres hybrides en gestation.

Projet front-end construit avec **React**, **TypeScript** et **Vite**, enrichi d'une configuration moderne incluant **ESLint (Flat Config)**, **Prettier** et **Vanilla Extract**.

---

## 🚀 Stack technique

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Vanilla Extract](https://vanilla-extract.style/)
- [ESLint (Flat config)](https://eslint.org/docs/latest/use/configure/)
- [Prettier](https://prettier.io/)

---

## 📦 Installation

```bash
git clone https://github.com/<votre-utilisateur>/les-productions-majeures.git
cd les-productions-majeures
npm install
```
